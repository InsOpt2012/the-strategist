# The Strategist — AI Life Coaching Tool

An AI-powered coaching session built on evidence-based 
behavioral science — ACT therapy, motivational interviewing, 
and polyvagal-informed regulation.

## What it does
Guides users through 8 coaching phases:
Regulation → Life Audit → Priorities → Energy → 
Decisions → Planning → Mindset → Synthesis

## Built with
- React + Vite
- Anthropic Claude API
- Vercel (hosting)

## To run locally
npm install
npm run dev

## To use the live app
Visit: https://the-strategist-one.vercel.app
You'll need a free Anthropic API key from console.anthropic.com
